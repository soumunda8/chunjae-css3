선택자

스타일을 부여할 대상을 지정하는 키워드

기본 선택자

태그, 클래스, 아이디, 전체 선택자

* {}      // 전체 선택자, 모든 태그
a {}      // a 태그로
.클래스명 {} // 태그에 해당 클래스속성이 있는 요소만 선택
#아이디명 {} // 태그에 해당 아이디속성이 있는 요소만 선택

클래스(class) : 스타일이나 스크립트를 공통으로 적용시키기 위한 방법

아이디(id) : 하나의 html문서에서 각 각의 요소를 주옵되지 않게 아이디를 부여하여 각 요소를 구분하여 스타일이나 스크립트를 각 각 적용하기 위해 태그에 지정

속성 선택자

태그이 속성에 다라 다르게 선택할 경우 사용

속성 존재 유무, 속성 일치, 속성 접두사, 속성 접미사를 포함하는 문자열

img[src] {}     // 이미지 중에서 src 속성이 존재하는 요소 선택
img[src=""] {}    // 이미지 중에서 src 속성의 값이 일치하는 요소 선택
img[src|="문자"] {} // 이미지 중에서 src 속성의 값이 특정 문자로 시작하는 요소를 선택
img[src^="문자"] {} // 이미지 중에서 src 속성의 값이 특정 문자로 시작하는 요소를 선택
img[src$="문자"] {} // 이미지 중에서 src 속성의 값이 특정 문자로 끝나는 요소 선택
img[src*="문자"] {} // 이미지 중에서 src 속성의 값이 특정 문자가 포함되는 요소 선택
img[src~="문자"] {} // 이미지 중에서 src 속성의 값이 특정 문자가 포함되는 요소 선택

링크 선택자

하이퍼링크 요소인 a요소의 현재 동작상태에 따라 달리 적용됨

link, visited 순으로 사용 / hover, active 순으로 사용해야 안 꼬임

a 요소에 href 속성의 값이 있는 경우 : link

a 요소에 href로 지정된 url이 방문한 적이 있는 경우 : visited

모든 요소에 마우스 포인터가 올라가는 경우 : hover [반응선택자]

모든 요소에 마우스를 누르고 있는 경우 : active [반응선택자]

a:link {}   //
a:visited {}  //
a:hover {}   //
a:active {}  //

선택영역 선택자 = ::selection
