배경 관련 속성

background-attachment(배경이미지를 고정할지, 스크롤하듯이 내려갈지 설정), background-blend-mode, background-clip, background-color, background-image, background-origin, background-position, background-repeat, background-size

background-color를 제외한 나머지 속성은 배경이미지에 대한 설정 속성임.

background : background-color background-image background-position / background-size background-repeat background-origin background-clip background-attachment

background 통합 속성은 여러 속성을 한 꺼번에 지정할 수 있지만, 각 세부 속성의 순서를 지키면서 설정해야 함. 특히 위치(position)과 크기(size)는 필수적으로 순서를 지켜야 함.



배경색(color)

background : rgba(221, 120, 18, 0.8);

background-color : rgba(221, 120, 18, 0.8);



그라디언트의 경우

background : linear-gradient(방향, 시작색, 종료색); -> background : linear-gradient(135deg, deeppink, deepskyblue)

background : radial-gradient(중심색, 중간색1, 중간색2, 중간색3, ..., 마지막색); -> background : radial-gradient(red, orange, gold, green, blue);



필터의 경우

filter : url(./images/bg01.webp);

filter : blue(5px);

filter : contrast(200%); -> 가까운건 밝고, 먼건 어둡게

filter : grayscale(80%); -> 100%는 흑백색

filter : hue-rotate(90deg);

filter : drop-shadow(10px 15px 20px red); -> drop-shadow(x축 y축 블러 그림자색상)



배경 이미지(image)

background-image : url(파일의 위치와 파일명);

background-image : url("./images/김기태.png"); -> 파일명이 한글일 경우나 중간에 띄어쓰기 있을 경우에는 쌍따옴표를 필수로, 아니면 자유



배경 위치(position)

background-position : 가로위치 세로위치;

background-position : 위치; -> 가로위치랑 세로위치가 동일한 경우

background-position : left center;
키워드일 경우 가로방향은 left/center/right, 세로방향은 top/center/bottom

background-position : 10% 100px;
특정 숫자가 기입될 경우는 단위를 반드시 명기를 해야함.

background-position : -200px -400px;
아이콘리스트이미지인 경우 해당 이미지의 이치 값에 따라 다른 아이콘이 나오게 할 수 있음 -> 스플릿이미지



배경 크기(size)

background-size : auto | length | cover | contain

auto : 배경이미지 크기 그대로 표시

length : %, px 등의 단위를 숫자와 그 숫자 뒤에 병기해서 사용

cover : 가로 폭의 맞추어 높이 부분은 감안하지 않는 방법

contain : 해당 콘텐츠의 크기에 맞추어 조절하는 방법 / 높이가 크면 높이에 맞추고, 폭이 크면 폭의 크기에 맞춤

background-size : 가로크기 세로크기;



배경 반복(repeat)

background-repeat : repeat(default) | repeat-x(가로 반복) | repeat-y(세로 반복) | no-repeat

해당 요소의 크기가 배경 이미지의 크기보다 클 경우에만 유효하다.

repeat : 가로 방향과 세로 방향 모두 적용됨

repeat-x : 가로 방향에만 적용됨

repeat-y : 세로 방향에만 적용됨

no-repeat : 반복하지 않고, 한 번만 출력



배경 시작점(origin)

background-origin : padding-box | border-box | content-box

배경 이미지가 어디부터 시작할지 결정

padding-box : padding 부분부터 이미지가 시작됨

border-box : border 부분부터 이미지가 시작됨

content-box : content 부분부터 이미지가 시작됨



배경 적용 범위(clip)

background-clip : padding-box | border-box | content-box

padding-box : padding 영역까지 이미지가 시작됨

border-box : border 영역까지 이미지가 시작됨

content-box : content 영역까지 이미지가 시작됨



배경 이미지 스크롤(attachment)

background-attachment : scroll(default) | fixed | local

배경 이미지가 마우스 스크롤시에서 스크롤 유무를 결정

scroll 또는 local : 일반 콘텐츠처럼 스크롤 됨

fixed : 배경 이미지가 화면에 고정되고, 스크롤 되지 않음
