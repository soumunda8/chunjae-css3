글자/문단 관련 속성

글자 관련 : color, font-size, font-weight, font-stretch, text-transform, font-variant, font-style, font-family, font-kerning, @font-face★

문단 관련 : letter-spacing, line-height(행간), letter-height, word-spacing, text-align, text-indent, text-decoration, text-jusify, text-orientation, text-overflow, white-space, word-break



★글자 색

color : 웹컬러명 | 16진코드 | rgb() / rgba() | hsl() / hsla()



★글자 크기

font-size : keyword | length

keyword : xx-small(가장 작은 글씨) | x-small | smaller(한 단계 더 더 작은 글씨) | small(한 단계 더 작은 글씨) | medium | large(한 단계 더 큰 글씨) | x-large | xx-large

length unit : px, em(rem), pt 등

keyword로 사용하여 글자 크기를 지정할 경우는 브라우저마다 모두 다르므로, length 값으로 지정하도록 하는 것이 좋다.

모노 관련 폰트는 색상 지정 불가 - 회색조



★글자 두께

font-weight : keyword | number

keyword : bolder | bold | normal | lighter

number : 100 ~ 900로 100단위로 기재. 100이 가장 가늘고, 900이 가장 두껍다.



글자 장평

font-stretch : utra-condensed | extra-condensed | condensed | semi-condensed | normal | semi-expanded | expanded | extra-expanded | ultra-expanded

expanded : 양적 확장 / extanded : 확장

장평(font-stretch)이란? 한 글자의 가로크기와 세로크기의 비율이며, 기준은 세로크기에 비해 가로크기가 얼마나 큰지에 대하여 값을 부여한다. / 한글 지원 X



★글자 스타일

font-style : normal | italic | oblique

기울임꼴의 유무를 나타내는 속성



글자 변형

font-variant : normal | small-caps

영문의 경우 대문자와 소문자의 크기가 각 각 달라서 균일하게 글자크기나 폭을 지정하기 힘들므로 대문자를 소문자 크기로 조절하기 위한 속성



글자 변환

text-transform : none | uppercase | lowercase | capitalize(첫 글자만 대문자)

해당 단어나 문장의 영문을 대문자 또는 소문자로 변환할 때 사용



★글꼴 지정

font-family : 1차폰트[, 2차폰트][, 3차폰트]

서버에 설치되어 있는 폰트의 이름을 지정하며, 만약 폰트의 이름 중간에 공백이 있거나 아니면, 한글과 같은 영문이 아닌 경우는 반드시 인용부호인 ""나 ''로 감싸서 표기해야 하며, 만약 1차 폰트가 존재하지 않을 경우는 2차 폰트가 적용되고, 여기서 2차 폰트도 적용될 수 없다면, 3차 폰트가 적용되게 된다. 다만, 3차 폰트가 없는 적용되지 못하는 경우를 대비하여 3차 폰트는 브라우저 기본적으로 제공하는 폰트를 지정하여야 함.

브라우저 제공하는 기본 폰트는 해당 서버나 크라이언트에 설치가 되어 있지 않아도 자동 제공되는데, sans-serif(고딕계열), serif(명조계열), cursive(궁서계열), monospace(장평이 같은 폰트), fantase(디자인 계열) 등을 말함.



커서 위치의 자간

font-kerning : auto | noraml | none



자간

letter-spacing : normal | length

length : px, em(rem), pt등의 단위를 주로 활용

해당 요소 안의 글자와 글자 사이의 간견



어간

word-spacing : normal | length

length : px, em(rem), pt등의 단위를 주로 활용

단어와 단어 사이의 간견



★행 높이 : 행간 - 블럭요소에만 적용

line-height : normal | number | length

length : px, em(rem), pt등의 단위를 주로 활용

1. 값으로 숫자만 기재할 경우에는 현재 문단의 글자크기의 배수가 된다. [ex]font-size:10px;line-height:5; 이면 행간은 50이 됨. 2. 숫자 뒤에 단위가 들어갈 경우는 해당하는 다뉘의 크기가 적용된다.

기본이 1.3이므로 font-size:10pt일때 3pt가 사이 간격이다.



★문단 정렬

text-align : left | center | right | justify(배분정렬 / 나눔정렬 - 글자사이가 자동으로 분배됨)

문단의 가로방향에 대한 정렬



★들여쓰기 / 내어쓰기

text-indent : length

length 단위 : px, %, pt

들여쓰기 : 다른 줄에 비해 첫 번째 줄이 더 들어가게 쓰는 것
text-ident : 20px

내어쓰기 : 다른 줄에 비해 첫 번째 줄이 더 나오게 쓰는 것
text-ident : -20px



★글자 장식

text-decoration : none | underline(a태그에서만 default) | overline | line-through

글자의 위, 중간, 아래에 줄을 표시하도록 하는 속성

none을 제외한 다른 값은 중복해서 사용할 수 있음.
text-decoration : overline underline;
text-decoration : underline double #369, overline wave #c00;

세부속성

text-decoration-color : 줄의 색

text-decoration-line : 줄의 위치(underline / overline / line-through)

text-decoration-style : 줄의 종류(solid / wave / double / dotted / dashed)



배분정렬

text-jusify : auto | noen(한글에서만 default) | inter-character(글자와 글자 사이가 멀어짐) | inter-word(단어와 단어 사이가 멀어짐)

단어 또는 글자 단위로 한 줄의 데이터 분량을 똑같이 배분정렬할 수 있는 속성



글자 진행 방향

text-orientation : mixed | upright | sideways | sideways-right | use-glyph-orientation(사용자가 쓰는 폰트에 맞추어서 읽음)



★글자 흘러 넘침

text-overflow : clip | ellipsis | 특정문자

text-overflow 는 overflow:hidden이며, white-space:nowrap인 경우에 유효하다.



★내용 보호 / 줄 끝 처리

white-space : normal | nowrap(무조건 한줄로 출력 / 자동줄바꿈 안함) | pre(엔터, 스페이스 그대로 출력) | pre-line(줄바꿈만 자동) | pre-wrap(줄바꿈 및 마지막에 엔터친 것도 자동)

하나의 단어가 한 줄에 입력되어야만 하기 때문에 그 때 한 줄에 단어가 배치가 되든지, 아니면 영문배치문법을 무시하고 단어 중간을 자를지, 또는 하나의 문장을 한 줄로만 표시할지 그 문장의 줄의 끝을 어떻게 처리할지 정하는 속성



단어 처리(영문에서만 거의 사용)

word-break : normal | break-all | keep-all(영문 배치를 무조건 지킴) | break-word(단어가 안 잘리게)

한 줄 또는 한 문장 안에서의 단어의 끝 처리 부분을 어떻게 할지 정하는 속성
